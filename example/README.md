# HoldYourBeer Flutter - Example & Test Apps

This directory contains alternative main entry points for testing different UI layouts and navigation styles.

## Files

### `main_simple.dart`
Simple test app for basic UI testing without full authentication flow.

**Run with:**
```bash
flutter run -t example/main_simple.dart
```

### `main_notched.dart`
Test app for notched bottom navigation bar style.

**Run with:**
```bash
flutter run -t example/main_notched.dart
```

### `main_inset.dart`
Test app for inset bottom navigation bar style.

**Run with:**
```bash
flutter run -t example/main_inset.dart
```

## Purpose

These files are for:
- UI/UX experimentation
- Testing different navigation patterns
- Development and prototyping
- Demonstrating alternative layouts

## Note

These are **not** production entry points. The main production app uses `/lib/main.dart`.
